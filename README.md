### FastAPI backend with Uniapp & Flutter frontend

#### Tech. stack:
Backend(Python):
- fastapi(http/websocket) + alembic
- sqlalchemy, motor
- dramatiq
- torch, transformers 
- AI models [ YOLOv8n, NLLB200, SDXL, gpt2-xl ]
- langchain for upper AI models

Frontend:
- uniapp(typescript + vue3)
- flutter(dart)
